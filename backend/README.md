<h2 align="center">Popote Backend</h2>

### Install

```bash
pipenv sync
```
Or, if you want to be able to run tests:
```bash
pipenv sync --dev
```


### Launching backend
```bash
python main.py
```
This will launch the backend on http://localhost:5000
You can see the OpenAPI documentation at http://localhost:5000/docs
